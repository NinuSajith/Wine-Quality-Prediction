# Wine-Quality-Prediction
Developed a predictive model for wine quality using supervised machine learning techniques.
Performed data preprocessing, feature engineering, and correlation analysis to identify key predictors.
Trained and optimized models including Random Forest and Neural Networks to improve performance.
Addressed class imbalance and improved model accuracy through tuning and validation techniques.
Evaluated models using metrics such as accuracy, precision, recall, and F1-score.
Generated actionable insights through data visualization and feature importance analysis.
